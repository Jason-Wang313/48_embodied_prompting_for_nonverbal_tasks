# Claims

Supported final claim:

- Nonverbal prompts should be represented as physical action contracts, not captions appended to text.
- Conservative embodied prompt graphs bind cue geometry, object or region, affordance, task phase, and safety state before action.
- In the full-scale suite, the conservative embodied prompt graph reaches 0.657 task accuracy, 0.828 safety-boundary recall, 0.516 cue-binding F1, and 0.534 utility.
- The unconservative graph reaches similar task accuracy but much higher unsafe execution and lower utility.

Bounded claim:

- The evidence is a deterministic synthetic diagnostic benchmark.
- Hardware perception, user adaptation, and certified safety are future validation layers.
- Safety-cue partial miss, cross-user style shift, and mobile manipulation remain difficult.
