# Hostile Prior Work

- Learning from demonstration already uses trajectories as instructions.
- Multimodal prompt papers already condition on image examples, language, and robot observations.
- Affordance-grounded language planners already filter infeasible commands.

Novelty therefore cannot be generic multimodal prompting. It must be the typed interface that maps gesture, gaze, contact, and pose traces to a robot action contract, including when to halt or ask for clarification.
