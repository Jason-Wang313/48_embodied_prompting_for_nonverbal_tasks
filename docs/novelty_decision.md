# Novelty Decision

Decision: final v3 full-scale submission artifact.

Best thesis:

Nonverbal robot prompts should be represented as conservative embodied prompt graphs that bind cue geometry, object or region, affordance, task phase, and safety state before action commitment.

Novelty boundary:

- Not a new gesture recognizer.
- Not a deployed human-robot safety system.
- Not generic multimodal prompting.
- Yes: a conservative action-contract interface with broad synthetic stress evidence and explicit failure boundaries.
