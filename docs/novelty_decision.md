# Novelty Decision

Decision after v2 hardening: workshop-only.

Best thesis: embodied prompt graphs can represent nonverbal prompts as physical task variables, including safety-critical cue bindings and clarification state.

Boundary:

- Not generic multimodal prompting.
- Not learning from demonstration.
- Not a deployed nonverbal perception system.
- Not a hardware safety claim.
- Not robust when safety cues are misbound or dropped.

The v2 binding-miss stress keeps the interface interesting but makes conservative cue binding the central requirement.
